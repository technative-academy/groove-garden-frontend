**Groove Garden** is a React-based frontend application that allows users to browse music, manage their own playlists, and build personal song collections.

---

## Features

- Browse playlists and songs fetched from a backend API
- Create, edit, and delete playlists
- Update playlist titles and descriptions
- Add/remove songs to/from a personal collection
- User authentication with JWT and refresh tokens
- Responsive UI designed with modular CSS

---

## Tech Stack

- **Frontend**: React, Redux Toolkit, React Router
- **Styling**: CSS Modules
- **API Layer**: Custom `apiService` abstraction
- **State Management**: Redux Toolkit
- **HTTP**: Fetch
- **Build Tool**: Create React App
